## 💻 Tecnologias

### Frontend
- **HTML5** - Estruturação semântica
- **CSS3** - Estilização e animações
  - Flexbox/Grid para layouts responsivos
  - Pré-processadores: SASS/SCSS (se aplicável)
- **JavaScript** - Interatividade e dinamismo
  - Framework: React/Vue/Angular (especifique)
  - Gerenciamento de estado: Redux/Vuex (se aplicável)

### Backend (se aplicável)
- **Node.js** / **PHP** / **Python** - Especifique sua stack
- **Banco de Dados**: MongoDB/MySQL/PostgreSQL
- **API**: RESTful / GraphQL

### Ferramentas e DevOps
- **Versionamento**: Git & GitHub
- **Deploy**: Vercel/Netlify/AWS
- **Testes**: Jest/Cypress (se aplicável)
